# Cantina 67

## Sobre
A Cantina 67 é um projeto simples de sistema/web para uma cantina, com foco em exibir produtos, cardápio e permitir pedidos.

## Funcionalidades
- Página inicial (Home)
- Cardápio
- Produtos
- Pedidos

## Tecnologias
- HTML
- CSS
- Python (Django)

## Estrutura
cantina67/
├── templates/
├── static/
├── app/

## Como rodar
1. Instalar o Django:
pip install django

2. Rodar o servidor:
python manage.py runserver

3. Acessar:
http://127.0.0.1:8000/

## Objetivo
Facilitar o acesso ao cardápio e organização dos pedidos.
